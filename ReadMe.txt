Typical hierarchy:
- HotKeys64_plugins - Plugins (Not for run applications. Used for send message to plugin and perform actions when call hotkey).
    - CapsSwitch64.dll
    - ChangeLayout64.dll
    - SendInput64.dll
    - VolumeControl64.dll
    - WindowControl64.dll
- HotKeys64.cfg - Configuration file.
- HotKeys64.exe - Core (Main non-GUI application).
- HotKeysSettings64.exe - Settings (Only to easy create configuration file for main application).
