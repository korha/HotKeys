Message:
%s% - One symbol. Print as is.
%string% - Insert string or emulate keydown or keyup. First and second symbols is service. First is used to open, second to close quoted down or up key name. String is unicode.

E.g.:
<>one<Enter<>Enter>two<Enter<>Enter>three
print:
one
two
three

XYXVolumeUpXYVolumeUpY
press VolumeUp key

ABabcdefghijklmnopqrstuvwxyz1234567890
only text, A and B ignored

ABABCDEFGHIJKLMNOPQRSTUVWXYZ1234567890
incorrect, because 3rd symbol A opening quote, but closing quote is missing.
